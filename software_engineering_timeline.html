Línea de Tiempo de la Ingeniería de Software

Explora los hitos más importantes que han definido nuestra disciplina

Cátedra: Ingeniería de Software | Alumno: Lopez Sardi Daniel

Comisión: 4K3 | Universidad: UTN - FRT | Año: 2025

Conceptos Fundamentales
1936
Máquina de Turing
Alan Turing introduce el concepto de "máquina universal", base teórica de toda software y computación moderna. Define qué problemas pueden ser resueltos algorítmicamente.
Impacto: Estableció los fundamentos matemáticos de la computación moderna, definió el concepto de algoritmo y sentó las bases para todos los desarrollos posteriores en ciencias de la computación.
Lenguajes de Programación
1945
Primer Programa Almacenado
John von Neumann describe la arquitectura de computadora de programa almacenado, donde instrucciones y datos comparten la misma memoria.
Impacto: Definió la arquitectura fundamental de las computadoras modernas, permitiendo la flexibilidad que hace posible el software actual y la programación tal como la conocemos.
Lenguajes de Programación
1948
Primer Lenguaje Ensamblador
Aparecen los primeros lenguajes de bajo nivel que permiten escribir programas en mnemónicos en vez de solo código máquina.
Impacto: Estableció el puente crucial entre el código máquina y los lenguajes de alto nivel, facilitando significativamente la programación y el desarrollo de software.
Lenguajes de Programación
1957
FORTRAN (IBM)
Primer lenguaje de alto nivel ampliamente usado, diseñado para cálculos científicos y numéricos.
Impacto: Demostró que era posible escribir programas eficientes en lenguajes de alto nivel, estableciendo el paradigma de traducción automática y abstracción en programación.
Lenguajes de Programación
1959
COBOL
Nace COBOL, orientado a negocios. Busca permitir la expansión del software en banca, gobierno y empresas.
Impacto: Demostró que los lenguajes podían ser específicos para dominios, estableció estándares para aplicaciones empresariales y sigue siendo usado en sistemas críticos.
Conceptos Fundamentales
1965
Multiprogramación
Se introducen sistemas operativos que permiten correr varios procesos a la vez, base para el multitasking.
Impacto: Revolucionó la eficiencia de las computadoras, permitiendo mejor utilización de recursos y sentando las bases para los sistemas operativos modernos.
Herramientas de Desarrollo
1969
UNIX
Ken Thompson y Dennis Ritchie desarrollan UNIX en Bell Labs. Piedra angular de los sistemas modernos.
Impacto: Estableció conceptos fundamentales como pipes, jerarquía de archivos y filosofía de herramientas pequeñas y especializadas. Base de Linux, macOS y sistemas modernos.
Lenguajes de Programación
1972
Lenguaje C
Dennis Ritchie crea C, que se convierte en el estándar para sistemas operativos y software de propósito general.
Impacto: Influenció prácticamente todos los lenguajes modernos, permitió el desarrollo de sistemas operativos portables y estableció estándares de programación de sistemas.
Herramientas de Desarrollo
1976
Microsoft y Apple
Microsoft vende su primer intérprete BASIC y Apple lanza su Apple I. El software empieza a comercializarse masivamente.
Impacto: Estableció la industria del software comercial y democratizó el acceso a las computadoras personales, transformando el software de herramienta académica a producto comercial.
Herramientas de Desarrollo
1983
GNU y el Movimiento del Software Libre
Richard Stallman crea el proyecto GNU y el concepto de licencias libres.
Impacto: Creó un ecosistema de herramientas de desarrollo libres, estableció principios éticos en el desarrollo y sentó las bases para el movimiento open source.
Herramientas de Desarrollo
1985
Windows 1.0
Microsoft lanza Windows, popularizando la interfaz gráfica de usuario (GUI).
Impacto: Democratizó el uso de computadoras personales, estableció estándares de interfaz gráfica y cambió la forma en que los usuarios interactúan con el software.
Herramientas de Desarrollo
1991
Linux y Python
Linus Torvalds libera el kernel Linux y Guido van Rossum presenta Python, ambos revolucionando el desarrollo y la comunidad open source.
Impacto: Linux se convirtió en la base de la infraestructura moderna de Internet, mientras Python democratizó la programación y se volvió esencial para IA y ciencia de datos.
Internet y Web
1995
Java y la Web
Sun Microsystems lanza Java con el lema "Write Once, Run Anywhere". El mismo año nace JavaScript junto con la web dinámica.
Impacto: Java estableció la portabilidad multiplataforma, mientras JavaScript se convirtió en el lenguaje universal de la web, transformando las páginas estáticas en aplicaciones interactivas.
Internet y Web
2004
Web 2.0
Surgen redes sociales, aplicaciones colaborativas y servicios en la nube. El software pasa a ser principalmente servicio (SaaS).
Impacto: Transformó el software de productos instalables a servicios en la nube, estableció nuevos modelos de negocio y cambió la forma en que consumimos y desarrollamos aplicaciones.
Conceptos Fundamentales
2010s
Inteligencia Artificial y Software en la Nube
Avances en IA, machine learning, microservicios y desarrollo en la nube. El software pasa a ser principalmente dinámico.
Impacto: Está redefiniendo completamente el desarrollo de software con automatización inteligente, arquitecturas distribuidas y nuevos paradigmas de programación asistida por IA.
Conceptos Fundamentales
2020s
IA Generativa en el Desarrollo
GitHub Copilot, ChatGPT y herramientas de IA comienzan a asistir directamente en la escritura de código, documentación y arquitectura de software.
Impacto: Está transformando la productividad del desarrollo, automatizando tareas repetitivas y permitiendo a los desarrolladores enfocarse en problemas de mayor nivel y creatividad.

Lenguajes de Programación
1980
C++ y Programación Orientada a Objetos
Bjarne Stroustrup comienza el desarrollo de C++, introduciendo la programación orientada a objetos al mundo del desarrollo de sistemas.
Impacto: Popularizó la OOP, permitiendo mejor organización del código, reutilización y mantenimiento de aplicaciones complejas. Influenció Java, C# y muchos otros lenguajes.
Herramientas de Desarrollo
1982
GNU y Software Libre
Richard Stallman lanza el Proyecto GNU, estableciendo los fundamentos del movimiento de software libre y las licencias copyleft.
Impacto: Creó un ecosistema de herramientas de desarrollo libres, estableció principios éticos en el desarrollo de software y sentó las bases para el desarrollo colaborativo.
Internet y Web
1990
World Wide Web
Tim Berners-Lee desarrolla la World Wide Web en el CERN, creando HTML, HTTP y URLs. Nace la era de las aplicaciones web.
Impacto: Transformó completamente el desarrollo de software, creando nuevos paradigmas de aplicaciones, arquitecturas distribuidas y la democratización del acceso a la información.
Herramientas de Desarrollo
1991
Linux - Sistema Operativo Libre
Linus Torvalds lanza Linux, un sistema operativo de código abierto que revoluciona el desarrollo colaborativo de software.
Impacto: Demostró el poder del desarrollo distribuido, estableció Linux como plataforma dominante en servidores e inspiró el modelo de desarrollo de código abierto.
Lenguajes de Programación
1995
JavaScript y Java
Brendan Eich crea JavaScript en Netscape, mientras Sun lanza Java. Dos lenguajes que definirán el desarrollo web y empresarial respectivamente.
Impacto: JavaScript se convierte en el lenguaje universal de la web, mientras Java establece la portabilidad ("write once, run anywhere") y la programación empresarial moderna.
Metodologías
2001
Manifiesto Ágil
17 desarrolladores firman el Manifiesto Ágil, revolucionando las metodologías de desarrollo con principios de colaboración, adaptabilidad y entrega continua.
Impacto: Transformó la industria del software, estableciendo metodologías como Scrum y XP, y cambiando el enfoque hacia el valor del cliente y la adaptabilidad.
Herramientas de Desarrollo
2005
Git - Control de Versiones Distribuido
Linus Torvalds crea Git para el desarrollo del kernel de Linux. Se convierte en el estándar de control de versiones distribuido.
Impacto: Revolucionó la colaboración en el desarrollo de software, permitió el desarrollo distribuido a gran escala y se convirtió en la base de plataformas como GitHub.
Internet y Web
2008
GitHub y Desarrollo Colaborativo
Se lanza GitHub, transformando la forma en que los desarrolladores colaboran y comparten código. Democratiza el desarrollo de software open source.
Impacto: Creó la mayor comunidad de desarrolladores del mundo, estableció nuevos estándares de colaboración y transparencia en el desarrollo de software.
Metodologías
2009
DevOps
Patrick Debois acuña el término "DevOps" en la conferencia DevOpsDays. Se establece la cultura de colaboración entre desarrollo y operaciones.
Impacto: Eliminó silos organizacionales, aceleró los ciclos de entrega, mejoró la calidad y confiabilidad del software mediante automatización e integración continua.
Herramientas de Desarrollo
2013
Docker y Contenedores
Docker populariza la tecnología de contenedores, revolucionando el deployment y la portabilidad de aplicaciones.
Impacto: Simplificó el deployment, mejoró la consistencia entre entornos, y habilitó arquitecturas de microservicios y cloud-native applications.
Conceptos Fundamentales
2015
Microservicios
Martin Fowler populariza el concepto de microservicios, estableciendo nuevos patrones arquitectónicos para aplicaciones distribuidas y escalables.
Impacto: Permitió mayor escalabilidad y flexibilidad en sistemas complejos, habilitó equipos independientes y aceleró la innovación en aplicaciones empresariales.
Conceptos Fundamentales
2018
Low-Code/No-Code
Las plataformas de desarrollo low-code y no-code ganan popularidad, democratizando el desarrollo de aplicaciones y permitiendo a no-programadores crear software.
Impacto: Expandió quién puede crear software, aceleró el desarrollo de aplicaciones empresariales y redujo la barrera de entrada para la innovación digital.
Conceptos Fundamentales
2020
IA en el Desarrollo de Software
GitHub Copilot y otras herramientas de IA comienzan a asistir en la escritura de código, marcando el inicio de la era de la programación asistida por IA.
Impacto: Está transformando la productividad del desarrollo, automatizando tareas repetitivas y permitiendo a los desarrolladores enfocarse en problemas de mayor nivel.
