Línea de Tiempo de la Ingeniería de Software

Explora los hitos más importantes que han definido nuestra disciplina

Cátedra: Ingeniería de Software | Alumno: Lopez Sardi Daniel

Comisión: 4K3 | Universidad: UTN - FRT | Año: 2025

Conceptos Fundamentales
1936
Máquina de Turing
Alan Turing introduce el concepto de "máquina universal", base teórica de la computación moderna.
Impacto: Fundamentos matemáticos de la computación y del concepto de algoritmo.
Lenguajes de Programación
1945
Programa almacenado
John von Neumann describe la arquitectura donde instrucciones y datos comparten memoria.
Impacto: Base de las computadoras modernas y la programación como la conocemos.
Lenguajes de Programación
1948
Primer lenguaje ensamblador
Surgen mnemónicos que reemplazan código máquina puro.
Impacto: Puente crucial entre código máquina y lenguajes de alto nivel.
Lenguajes de Programación
1957
FORTRAN (IBM)
Lenguaje de alto nivel orientado a cálculos científicos.
Impacto: Prueba que los lenguajes de alto nivel pueden ser eficientes.
Lenguajes de Programación
1959
COBOL
Lenguaje orientado a negocios para banca, gobierno y empresas.
Impacto: Estandariza aplicaciones empresariales; aún en sistemas críticos.
Conceptos Fundamentales
1965
Multiprogramación
Sistemas operativos capaces de ejecutar varios procesos a la vez.
Impacto: Mejor uso de recursos; base de los SO modernos.
Herramientas de Desarrollo
1969
UNIX
Thompson y Ritchie desarrollan UNIX en Bell Labs.
Impacto: Pipes, jerarquía de archivos y filosofía de herramientas pequeñas.
Lenguajes de Programación
1972
Lenguaje C
Dennis Ritchie crea C, estándar para SO y software general.
Impacto: Influye en casi todos los lenguajes modernos.
Herramientas de Desarrollo
1976
Microsoft y Apple
Microsoft vende BASIC; Apple lanza Apple I.
Impacto: Nace la industria del software comercial.
Lenguajes de Programación
1980
C++ y Programación Orientada a Objetos
Bjarne Stroustrup inicia C++ e introduce OOP a sistemas.
Impacto: Populariza la OOP; base para Java, C# y más.
Herramientas de Desarrollo
1983
GNU y movimiento de software libre
Richard Stallman lanza el proyecto GNU y licencias libres.
Impacto: Bases del open source y ecosistema de herramientas libres.
Herramientas de Desarrollo
1985
Windows 1.0
Microsoft populariza la GUI.
Impacto: Democratiza el uso de PCs y define la interacción gráfica.
Internet y Web
1990
World Wide Web
Tim Berners-Lee crea HTML, HTTP y las URLs en el CERN.
Impacto: Nace la era de las aplicaciones web.
Herramientas de Desarrollo
1991
Linux y Python
Torvalds libera el kernel Linux; Guido presenta Python.
Impacto: Linux sostiene Internet; Python democratiza la programación e IA.
Internet y Web
1995
Java y JavaScript
Sun lanza Java; Netscape presenta JavaScript y la web dinámica.
Impacto: Portabilidad “write once, run anywhere” y el lenguaje universal de la web.
Metodologías
2001
Manifiesto Ágil
17 desarrolladores firman los principios de agilidad.
Impacto: Cambia el foco a valor, colaboración y adaptación.
Internet y Web
2004
Web 2.0
Redes sociales, colaboración y servicios cloud/SaaS.
Impacto: Del producto instalable al servicio en la nube.
Herramientas de Desarrollo
2005
Git
Torvalds crea Git, estándar del control de versiones distribuido.
Impacto: Colaboración masiva y base de GitHub.
Internet y Web
2008
GitHub
Plataforma que transforma la colaboración y el open source.
Impacto: Mayor comunidad de desarrolladores y transparencia.
Metodologías
2009
DevOps
Cultura que integra desarrollo y operaciones.
Impacto: Entrega continua, calidad y confiabilidad.
Herramientas de Desarrollo
2013
Docker y contenedores
Populariza contenedores y portabilidad de apps.
Impacto: Consistencia entre entornos; base cloud-native.
Conceptos Fundamentales
2015
Microservicios
Patrón para apps distribuidas y escalables.
Impacto: Equipos autónomos y despliegue independiente.
Conceptos Fundamentales
2018
Low-Code / No-Code
Plataformas que democratizan el desarrollo.
Impacto: Acelera apps empresariales y baja la barrera de entrada.
Conceptos Fundamentales
2020s
IA generativa en el desarrollo
Copilot, ChatGPT y otras asisten en código, docs y arquitectura.
Impacto: Automatiza tareas y eleva el foco creativo.
